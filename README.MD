## Installation

```sh
npm install @vodyani/winston
```

## Recommendations

- typescript `v4.6.x`
- node.js `v12.x` ~ `v16.x`

## Dependencies

- [winston-daily-rotate-file](https://github.com/winstonjs/winston-daily-rotate-file) `v4`
- [winston](https://github.com/winstonjs/winston) `v3`
